# Barber Shop Appointment Booking System – Frontend

## Installation & Running

### 1. Install dependencies

```bash
npm install
```

### 2. Development

```bash
npm run dev
```

Frontend runs at http://localhost:3000

### 3. Production build

```bash
npm run build
npm start
```

## Docker

```bash
docker build -t barber-frontend .
docker run -p 3000:3000 barber-frontend
```
